🆕 新增 4 个数据结构

1. DocumentTemplate (单证模板)
id, doc_type (发票/提单/信用证...)
name, description
template_content (HTML 模板)
fields_config (JSON: 字段定义/校验规则/默认值)
is_system (系统模板 vs 自定义)
created_by → User, course_id → Course
is_enabled, version, created_at, updated_at
用途：教师可以自定义单证格式，调整字段要求，适应不同教学需求
2. ProductionOrder (工厂生产单)
id, order_no, transaction_id → Transaction
factory_id → Company, exporter_id → Company
product_id → Product, quantity, unit_price, amount
delivery_date, production_status (待生产/生产中/已发货)
actual_delivery_date, notes, created_at, updated_at
用途：工厂角色接收出口商订单，安排生产、发货，完成供货流程
3. ScoringRule (评分规则配置)
id, course_id → Course
rule_type (交易完成/单证准确/时效性/成本控制...)
weight (权重), max_score
config (JSON: 具体评分逻辑)
is_enabled, sort_order, created_at, updated_at
示例 config：{"on_time_bonus": 10, "error_penalty": 5, "days_limit": 30}
用途：教师可自定义评分权重和规则，灵活调整考核重点
4. News (新闻公告)
id, course_id → Course
title, content, publish_type (新闻/公告/通知)
priority (普通/重要/紧急), is_top
published_by → User, published_at
status (草稿/已发布/已撤回)
view_count, created_at, updated_at
用途：教师发布市场动态、政策变化、操作指南等，辅助教学
