8 个核心状态机 + 联动关系

📊 状态机总览

1. Transaction (交易)
草稿 → 询盘 → 还盘
→ 待签约 → 已签约
→ 履约中 → 已完成
2. Contract (合同)
草稿 → 待签字
→ 双方已签字
→ 已生效 → 履行完毕
3. Document (单证)
草稿 → 待审核 → 已审核
→ 已提交 → 已接收 → 已归档
4. LetterOfCredit (信用证)
草稿 → 待开证 → 已开证
→ 已通知 → 待议付
→ 已议付 → 已付款
5. ProductionOrder (生产单)
待确认 → 已确认
→ 生产中 → 已完成
→ 已发货
6. Payment (付款)
待支付 → 处理中
→ 已支付 → 已收款
7. Course (课程)
筹备中 → 报名中
→ 进行中 → 已结束
→ 已归档
8. Score (成绩)
计算中 → 待审核
→ 已发布 → 已确认

🔗 状态机联动关系

自动触发规则
# 合同生效 → 触发信用证流程
if contract.status == EFFECTIVE:
    if contract.payment_term == 'L/C':
        LetterOfCredit.objects.create(contract=contract)
# 信用证通知 → 交易履约
if lc.status == ADVISED:
    lc.transaction.status = IN_PROGRESS
# 所有单证归档 → 交易完成
if all_docs_archived(transaction):
    transaction.status = COMPLETED
# 课程结束 → 触发成绩计算
if course.status == ENDED:
    for student in course.students:
        Score.calculate(student, course)
# 所有学生确认 → 课程归档
if all_scores_confirmed(course):
    course.status = ARCHIVED
