🆕 新增数据结构详解

1. 财务费用系统
TransactionCost (交易费用)
id, transaction_id → Transaction
cost_type (货款/运费/保费/报关费/银行费/检验费/其他)
amount, currency, payer_id → Company, payee_id → Company
status (待支付/已支付), paid_at, created_at
Payment (收付款记录)
id, cost_id → TransactionCost
amount, payment_method (L/C/T/T/D/P/D/A)
bank_ref, status, created_at
2. 系统配置数据
ExchangeRate (汇率)
id, from_currency, to_currency
rate, effective_date, course_id → Course (可课程级覆盖)
ShippingRate (运费率)
id, from_port_id → Port, to_port_id → Port
carrier, rate_per_unit (per CBM/KG), currency
transit_days, effective_date
InsuranceRate (保险费率)
id, coverage_type (一切险/平安险/水渍险)
rate_premium, deductible, effective_date
3. 地理数据
Country (国家)
id, code (CN/US/JP...), name (中文/英文)
currency (CNY/USD/JPY...), calling_code
is_enabled, sort_order
Port (港口)
id, code (SHA/NYC/TYO...), name (中文/英文)
country_id → Country, port_type (海港/空港/陆港)
is_enabled, sort_order
4. 消息通知
Message (消息)
id, from_user_id → User, to_user_id → User
course_id → Course
type (系统通知/交易消息/单证消息/询盘/其他)
title, content, related_entity_type, related_entity_id
is_read, read_at, created_at
5. 单证依赖关系
DocumentDependency (单证依赖)
id, doc_type, depends_on_doc_type
dependency_type (precedes/synchronous/follows)
description (如：发票日期≤提单日期)
