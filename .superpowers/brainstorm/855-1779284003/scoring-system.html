📐 评分公式

总分计算公式
总分 = 盈利能力×W1 + 单证准确×W2 + 完成时效×W3 + 业务量×W4 + 综合表现×W5
1. 盈利能力 (Profit Score)
= Σ(每笔交易利润 / 每笔交易金额) × 100 / 交易笔数
2. 单证准确 (Accuracy Score)
= 100 - Σ(单证错误数 × 错误权重) / 单证总数
3. 完成时效 (Efficiency Score)
= Σ(基准分 - 超时天数×扣分) / 交易笔数
4. 业务量 (Volume Score)
= MIN(完成交易笔数 / 目标笔数 × 100, 上限分)
5. 综合表现 (Overall Score)
= 规范操作加分 - 违规操作扣分

⚙️ 评分规则配置 (ScoringRule)

教师可配置参数
参数 默认值 说明
profit_weight 30% 盈利能力权重
accuracy_weight 25% 单证准确权重
efficiency_weight 20% 完成时效权重
volume_weight 15% 业务量权重
overall_weight 10% 综合表现权重
doc_error_penalty 5分/个 单证错误扣分
overdue_penalty 2分/天 超时扣分
target_transactions 3笔 目标交易笔数
time_limit_days 30天 完成时限

📊 单证错误检测

自动校验规则
校验项 规则 扣分
日期逻辑 发票日期 ≤ 提单日期 ≤ 保险日期 10分
金额一致 发票金额 = 汇票金额 = 保险金额×1.1 15分
数量一致 发票数量 = 装箱单数量 = 提单数量 10分
单证齐全 L/C 要求的单证全部提交 每缺一种 20分
必填字段 所有必填字段不能为空 每空一项 2分
背书签字 需要背书/签字的单证已签字 每漏一项 5分
