为什么兼容性这么好？

服务端渲染（SSR）输出的是标准 HTML，这是所有浏览器都能理解的基础语言。

jQuery处理了浏览器之间的差异，让你写一次代码就能在所有浏览器运行。

✅ 优点

  • 零门槛 — 所有浏览器原生支持，无需任何配置
  • 学校机房友好 — 无论什么配置、什么系统都能用
  • 维护简单 — 不需要处理打包工具、babel 转译等
  • 性能可预测 — 服务器压力大但客户端轻量

⚠️ 权衡

  • 页面切换会刷新（传统 Web 方式）
  • 复杂交互需要较多 JavaScript 代码
  • 用户体验不如现代 SPA 应用流畅

💡 推荐：Django + Jinja2 + jQuery

这是经过时间考验的组合，非常适合教育实训系统：

  • Django 内置管理后台 — 教师管理非常方便
  • Django ORM — 快速处理业务逻辑
  • 丰富的第三方库 — 用户认证、权限、报表等都有现成方案
