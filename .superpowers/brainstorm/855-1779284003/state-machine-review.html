需要评估的业务实体
业务实体 是否需要状态机 原因

🆕 建议补充的 3 个状态机

1. Contract (合同) 状态机
草稿 → 待签字 → 双方已签字 → 已生效 → 履行完毕
合同需要双方签字，签字后生效，是外贸业务的核心单据
2. Course (课程) 状态机
筹备中 → 报名中 → 进行中 → 已结束 → 已归档
课程状态控制学生能否进入系统、提交作业等
3. Score (成绩) 状态机
计算中 → 待审核 → 已发布 → 已确认
成绩需要教师审核后发布，学生确认后最终归档

🔗 状态机联动关系

状态间的自动触发
合同 → 信用证
Contract.已生效 → 自动创建 LetterOfCredit.草稿
信用证 → 交易履约
LetterOfCredit.已通知 → Transaction.履约中
交易 → 课程状态
全部交易完成 → Course.可结束
课程结束 → 成绩计算
Course.已结束 → 触发 Score.计算中
