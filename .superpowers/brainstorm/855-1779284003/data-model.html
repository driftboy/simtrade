数据模型设计

核心数据结构

用户与角色

用户 - 角色关系 (多对多)
User (用户) div>
id, username, password, name, student_id, email
class_id → Class, is_active, created_at
Role (角色类型) div>
id, code, name, is_playable, is_enabled_by_default
codes: EXPORTER, IMPORTER, FACTORY, BANK, CUSTOMS, SHIPPING...
UserRole (用户角色分配) div>
id, user_id → User, role_id → Role, company_id → Company
course_id → Course, created_at

公司与商品

基础业务数据
Company (公司) div>
id, name, country, city, address, role_type
capital, business_scope, owner_id → User
Product (商品) div>
id, name, category, spec, unit, image_url
cost_price, reference_price, min_order_qty
Catalog (商品目录) div>
id, company_id → Company, product_id → Product
price, stock, is_available

交易流程

交易核心数据
Transaction (交易) div>
id, transaction_no, type, status, course_id → Course
buyer_id → Company, seller_id → Company
product_id → Product, quantity, unit_price, amount
trade_term, payment_term, shipment_term
created_at, updated_at
Inquiry (询盘) div>
id, transaction_id → Transaction, from_id, to_id
message, status, created_at
Contract (合同) div>
id, contract_no, transaction_id → Transaction
content, signed_at, status
LetterOfCredit (信用证) div>
id, lc_no, contract_id → Contract, amount
issuing_bank_id, advising_bank_id
issue_date, expiry_date, status

单证系统

单证数据
Document (单证) div>
id, doc_no, doc_type, transaction_id → Transaction
template_id → Template, content, status
created_by → User, verified_by, verified_at
DocumentTemplate (单证模板) div>
id, doc_type, name, template_content, fields_config
is_enabled

教学管理

课程与评分
Course (课程/实习计划) div>
id, name, code, teacher_id → User
start_date, end_date, settings (JSON)
role_settings (JSON) — 角色开关配置
ActivityLog (操作日志) div>
id, user_id → User, course_id → Course
action, entity_type, entity_id, details (JSON)
ip_address, created_at
Score (成绩) div>
id, user_id → User, course_id → Course
total_score, detail_scores (JSON)
rank, updated_at
