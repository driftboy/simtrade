# Admin 导航重构设计文档

**日期**: 2026-06-02
**状态**: 待审核
**版本**: 1.0

## 概述

统一管理后台入口，添加侧边导航栏，优化页面布局，提升管理员使用体验。

## 当前问题

1. 存在两个功能重叠的"管理后台"入口 (`/admin-panel/` 和 `/dashboard/admin`)，造成混淆
2. URL 不一致，部分快捷链接指向不同入口
3. 缺少侧边导航栏，用户需通过面包屑或返回按钮导航
4. 系统设置页面过于密集，缺乏清晰的视觉分组

## 设计目标

1. 统一管理后台入口为 `/admin/`
2. 添加固定左侧导航栏
3. 添加面包屑导航
4. 优化系统设置页面布局

## URL 结构

```
/admin/                    - 概览（统计卡片 + 图表 + 最近活动）
/admin/users/              - 用户管理
/admin/system/             - 系统设置（汇率、HS编码、海关参数）
```

以下模块保持现有路径，通过侧边导航访问：
- `/teaching/` - 教学管理（课程、班级、学期）
- `/documents/` - 单证管理
- `/companies/` - 公司管理

## 侧边导航菜单

| 菜单项 | 图标 | 路径 | 说明 |
|--------|------|------|------|
| 概览 | glyphicon-home | /admin/ | 统计数据、图表、最近活动 |
| 用户管理 | glyphicon-user | /admin/users/ | 用户列表、角色修改、密码重置 |
| 教学管理 | glyphicon-book | /teaching/ | 课程、班级、学期管理 |
| 单证管理 | glyphicon-file | /documents/ | 单证列表 |
| 公司管理 | glyphicon-briefcase | /companies/ | 公司列表 |
| 系统设置 | glyphicon-cog | /admin/system/ | 汇率、HS编码、海关参数 |

## 布局结构

```
┌─────────────────────────────────────────────┐
│ 顶部导航栏（复用现有 base.html）            │
├──────────┬──────────────────────────────────┤
│          │ 面包屑导航                       │
│          ├──────────────────────────────────┤
│  侧边    │                                  │
│          │         主内容区域               │
│          │                                  │
│  导航    │         （各页面具体内容）        │
│          │                                  │
│          │                                  │
└──────────┴──────────────────────────────────┘
```

### 侧边导航规格

- **宽度**: 200px
- **位置**: 固定在左侧
- **样式**: 深色背景 (#262626)，当前页高亮 (#4a90d9)
- **菜单项**: 图标 + 文字，hover 效果

### 面包屑导航

- **位置**: 主内容区顶部
- **格式**: `首页 > 管理后台 > 当前页面`
- **样式**: Bootstrap breadcrumb

## 页面迁移计划

### 需要修改的文件

| 文件 | 操作 | 说明 |
|------|------|------|
| `simtrade/urls.py` | 修改 | 将 `/admin-panel/` 改为 `/admin/` |
| `templates/admin_panel/dashboard.html` | 重构 | 添加侧边导航，调整布局 |
| `templates/admin_panel/user_list.html` | 重构 | 添加侧边导航 |
| `templates/admin_panel/system.html` | 重构 | 添加侧边导航，优化布局 |
| `templates/dashboard/admin.html` | 合并或废弃 | 合并图表到概览页面 |
| `static/css/dashboard.css` | 新增 | 侧边导航样式 |
| `templates/base.html` | 修改 | 更新导航链接 |

### 系统设置页面优化

保持现有 3 个模块（汇率、HS编码、海关参数），仅优化：
- 面板间距统一
- 标题样式统一
- 表格样式统一
- 按钮组对齐

## 交互行为

1. **侧边导航状态管理**: 使用 JavaScript 高亮当前页菜单项
2. **面包屑自动生成**: 根据当前路径自动生成面包屑
3. **响应式支持**: 小屏幕下侧边导航可收起
